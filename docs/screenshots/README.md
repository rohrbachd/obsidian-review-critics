# Screenshot Guide

Use this folder for publication/demo screenshots referenced by the root `README.md`.

## Recommended Files

- `commands-palette.png`  
  Command palette showing plugin commands.
- `live-preview-tokens.png`  
  A note in Live Preview with add/delete/highlight/replace tokens visible.
- `comments-pane.png`  
  Open comments pane with several comment entries.
- `settings-tab.png`  
  Plugin settings with author and color options.

## Capture Tips

- Use an example note with realistic text and no private data.
- Keep Obsidian UI language in English for reviewer clarity.
- Prefer width around 1200-1600px for desktop screenshots.
- Crop to relevant UI so images remain readable in GitHub markdown.
